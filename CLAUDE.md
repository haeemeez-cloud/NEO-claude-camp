# CLAUDE.md

## 프로젝트 소개

패션 플랫폼 상품 관리 프로젝트.
MD(Merchandiser)가 상품 데이터와 매출 데이터를 분석하고 관리한다.

## 사용자 환경

- OS: Windows
- 셸: PowerShell / Git Bash
- 경로 구분자: 백슬래시(`\`) 사용 (Windows 기준)

## 데이터 규칙

- **브랜드명**: 한글로 표기
- **가격**: 원 단위, 천 단위 쉼표 표기 (예: 128,000원)
- **시즌 표기**: `25SS`, `24FW` 형식
- **SKU**: 대문자 + 숫자 조합 (예: `TOP-001`)
- **날짜 형식**: `YYYY-MM-DD`

## 주요 데이터 파일

| 파일 | 내용 |
|------|------|
| `data/products.csv` | 상품 마스터 (SKU, 브랜드, 카테고리, 가격, 재고, 시즌) |
| `data/sales.csv` | 일별 매출 (주문일, SKU, 수량, 매출, 반품) |
| `data/brands.csv` | 입점 브랜드 (수수료율, 계약기간, 담당자) |

## 작업 스타일

- 데이터 분석 결과는 표(table) 형태로 먼저 보여준다
- 코드보다 결과 해석을 중심으로 설명한다
- CSV 작업 시 한글 인코딩은 `UTF-8-BOM` 또는 `CP949`를 확인한다
