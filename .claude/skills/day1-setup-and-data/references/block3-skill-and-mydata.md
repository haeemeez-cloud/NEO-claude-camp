# Block 3: 스킬 만들기 + 내 데이터 도전

> Block 2에서 한 전처리 작업을 Skill로 만들어서, 다음번에 한 줄로 실행할 수 있게 한다.

> **Phase A 시작 시 반드시 아래 형태로 출력한다:**
> ```
> 📖 공식 문서 (Skill): https://docs.anthropic.com/en/docs/claude-code/skills
> ```

## EXPLAIN

### 왜 스킬로 만드나요?

Block 2에서 했던 작업을 떠올려보자:

1. 데이터 구조 파악
2. 카테고리명 통일
3. 날짜 형식 통일
4. 수수료율 형식 통일
5. 3개 파일 합쳐서 보고서 생성

이 작업을 **매번 처음부터 설명**해야 한다면 비효율적이다. 스킬로 만들면:

```
전: "products.csv에서 카테고리명을 통일하고, sales.csv에서 날짜를... (5줄 설명)"
후: "/data-cleanup"  (한 줄)
```

> 비유: 요리를 처음 할 때는 레시피를 보면서 하지만, 레시피를 저장해두면 다음부터 제목만 보고 바로 만들 수 있다. 스킬 = 저장된 레시피.

### 스킬의 구조 복습

```
.claude/skills/data-cleanup/
└── SKILL.md          ← 이 파일 하나만 있으면 스킬이 된다
```

SKILL.md 파일 안에는 2가지가 들어간다:

```markdown
---
name: data-cleanup
description: CSV 데이터 전처리. 카테고리명 통일, 날짜 형식 통일, 수수료율 통일, 보고서 생성.
---

# 데이터 전처리 스킬

(여기에 Claude가 따라야 할 작업 순서를 적는다)
```

- `---` 사이의 영역: **frontmatter**. 스킬의 이름과 설명을 적는 곳
- 그 아래: **본문**. Claude가 실제로 수행할 작업 내용

### 전처리 스킬이란?

"이 형식의 데이터가 들어오면 이렇게 정리해줘"를 문서로 만든 것이다.

포함할 내용:
- 어떤 파일을 대상으로 하는지
- 어떤 전처리를 할 것인지 (카테고리 통일, 날짜 통일 등)
- 최종 산출물은 무엇인지 (정제된 CSV? 보고서?)

## EXECUTE

아래 2단계를 순서대로 Claude Code에게 요청해보세요:

### 1단계: 스킬 만들기

Claude Code 대화창에 아래를 입력한다:

```
방금 한 CSV 전처리 작업을 스킬로 만들어줘.

스킬 이름: data-cleanup
저장 위치: .claude/skills/data-cleanup/SKILL.md

스킬 내용:
1. data/ 폴더의 CSV 파일들을 읽고 구조와 문제점을 분석
2. 카테고리명 한글 통일 (OUTER→아우터 등)
3. 날짜 형식 YYYY-MM-DD 통일
4. 수수료율 소수점 형식 통일
5. 중복 행 제거
6. 빈 값 확인 및 보고
7. 정제된 데이터로 브랜드별 매출 요약 보고서 생성

다음번에 /data-cleanup 하면 같은 전처리를 자동으로 할 수 있게 해줘.
```

> Claude가 SKILL.md 파일을 만들어준다. 파일 내용을 읽어보고, 빠진 것이 있으면 수정을 요청하자.

### 2단계: 스킬 테스트

새 세션을 열고(기존 대화를 닫고 `claude`를 다시 실행) 아래를 입력한다:

```
/data-cleanup
```

> 스킬이 실행되면서 Block 2에서 했던 전처리 작업을 자동으로 수행한다. 제대로 동작하는지 확인하자.

### 내 데이터 도전

```json
AskUserQuestion({
  "questions": [{
    "question": "혹시 본인이 실제 업무에서 쓰는 CSV/엑셀 데이터가 있나요?\n있으면 같이 정리해볼 수 있어요!",
    "header": "내 데이터로 실습",
    "options": [
      {"label": "있어, 해볼래!", "description": "내 데이터를 Claude에게 정리시켜보고 싶다"},
      {"label": "지금은 괜찮아", "description": "오늘은 샘플 데이터로 충분했다"},
      {"label": "나중에 해볼게", "description": "데이터는 있는데 지금 당장은 안 됨"}
    ],
    "multiSelect": false
  }]
})
```

#### "있어, 해볼래!" 선택 시:

아래 순서로 안내한다:

1. "그러면 데이터 파일(CSV 또는 엑셀)을 이 프로젝트 폴더에 넣어주세요."
2. "파일을 넣었으면 '완료'라고 입력해주세요."
3. 사용자가 "완료"라고 하면:
   - 프로젝트 폴더에 새로 추가된 CSV/엑셀 파일을 찾는다
   - 파일을 읽고 구조와 문제점을 분석한다
   - 어떤 전처리가 필요한지 제안한다
   - 사용자 동의 후 전처리를 실행한다
   - (선택) 이 전처리도 스킬로 만들 수 있다고 안내한다

> ⚠️ 주의: 사용자의 실제 데이터에는 민감한 정보가 포함될 수 있다. 데이터 내용을 외부에 공유하지 않도록 안내한다.

#### "지금은 괜찮아" 또는 "나중에 해볼게" 선택 시:

다음에 본인 데이터로 실습하고 싶으면, 데이터 파일을 프로젝트 폴더에 넣고
Claude Code에게 "이 데이터를 분석하고 정리해줘"라고 요청하면 됩니다.
```
