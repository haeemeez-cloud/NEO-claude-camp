> **Phase A 시작 시 반드시 아래 형태로 출력한다:**

> 📖 참고: CSV 파일은 .claude/skills/day1-setup-and-data/data/ 폴더에 있습니다.

## EXPLAIN

### CSV가 뭔가요?

CSV는 **Comma-Separated Values**의 약자다. 쉼표로 데이터를 구분하는 텍스트 파일이다.

```
이름,나이,직업
홍길동,30,개발자
김영희,28,디자이너
```

> 비유: 엑셀의 텍스트 버전. 엑셀 파일을 "다른 이름으로 저장" → "CSV" 형식으로 저장하면 CSV가 된다.

### 전처리가 뭔가요?

전처리는 **데이터를 분석하기 전에 깨끗하게 정리하는 작업**이다.

현실의 데이터는 완벽하지 않다:
- 같은 것을 다른 이름으로 적음 (예: "아우터" vs "OUTER")
- 빈 칸이 있음 (재고 수량 누락)
- 중복 데이터가 있음 (같은 SKU가 2번 등장)
- 날짜 형식이 제각각 (예: "2025-01-01" vs "25/01/01")

> 비유: 보고서를 작성하기 전에 원본 데이터를 정리하는 것. 엑셀에서 필터 걸고, 오타 수정하고, 빈 칸 채우는 그 작업을 Claude에게 시킨다.

### Claude Code에게 데이터 작업을 시키는 법

Claude Code에게 데이터 작업을 시킬 때는 **자연어(평소 말하는 말)**로 요청하면 된다.

```
❌ 안 좋은 요청: "데이터 정리해줘"
✅ 좋은 요청: "products.csv에서 카테고리 이름이 영어인 것을 한글로 바꿔줘. OUTER는 아우터로."
```

핵심: **원하는 결과를 구체적으로 설명**할수록 더 정확한 결과가 나온다.

### 오늘 사용할 샘플 데이터 3종

| 파일 | 내용 | 데이터 건수 | 의도적으로 넣은 문제점 |
|------|------|------------|----------------------|
| `data/products.csv` | 상품 마스터 | ~20행 | 카테고리명 불일치(아우터/OUTER), 빈 값, 중복 SKU |
| `data/sales.csv` | 일별 매출 | ~30행 | 날짜 형식 혼재(YYYY-MM-DD / YY/MM/DD), 누락 SKU |
| `data/brands.csv` | 브랜드 관리 | ~10행 | 수수료율 형식 혼재(15% / 0.15), 날짜 형식 불일치, 중복 행 |

> 이 데이터들은 실습용으로 만든 것이다. 실제 패션 플랫폼 MD가 다루는 데이터와 비슷한 구조로 만들었다.

## EXECUTE

아래 3단계를 순서대로 Claude Code에게 요청해보세요:

### 1단계: 데이터 파악

Claude Code 대화창에 아래를 입력한다:

```
.claude/skills/day1-setup-and-data/data/ 폴더에 있는 CSV 파일들을 읽고,
각 파일의 구조와 문제점을 분석해줘.
```

> Claude가 각 파일을 읽고 컬럼 구조, 데이터 타입, 발견된 문제점을 정리해준다. 어떤 문제를 찾아내는지 확인해보자.

### 2단계: 데이터 정제

Claude Code에게 다음 3가지를 요청한다 (한 번에 모아서 요청해도 되고, 하나씩 요청해도 된다):

```
방금 분석한 CSV 데이터를 정리해줘:
1. products.csv에서 카테고리명을 통일해줘 (OUTER→아우터 등)
2. sales.csv에서 날짜 형식을 YYYY-MM-DD로 통일해줘
3. brands.csv에서 수수료율을 소수점 형식(0.15)으로 통일해줘
```

> Claude가 Python 코드를 작성해서 자동으로 데이터를 정리한다. 수정된 결과를 확인해보자.

### 3단계: 데이터 통합

Claude Code에게 아래를 요청한다:

```
정리된 3개 CSV를 합쳐서 '브랜드별 1월 매출 요약 보고서'를 만들어줘.
포함할 내용:
- 브랜드명
- 총매출
- 판매건수
- 반품률
- 수수료율
- 수수료 차감 후 정산금액
```

> Claude가 3개 파일을 조인(join)해서 보고서를 만들어준다. 엑셀에서 VLOOKUP + 피벗 테이블로 할 작업을 자연어 한 줄로 한 것이다.
